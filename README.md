# Fake News Generator & Detector

A Python-based project that uses NLP and machine learning to generate and detect fake news. Includes an interactive Streamlit web interface, TF-IDF feature extraction, and Naive Bayes classification.

# Highlights

* Fake news detection with machine learning
* Fake news generation capability
* Streamlit-powered interactive UI
* Built with Python, scikit-learn, TF-IDF, and Naive Bayes

# Tech Stack
* Python
* Scikit-learn
* TF-IDF
* Naive Bayes
* Streamlit

# Features
Fake news generator
Fake news detection
Interactive web interface
